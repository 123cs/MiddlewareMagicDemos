Camel ActiveMQ Project
======================

This project Shows how to connect to the Apache ActiveMQ broker (default broker of WildFly10) from a standalone Apache Camel Route and send some XML messages to the JMS Queue.


To build this project use
————
    mvn install


To run this project use the following Maven goal
————
    mvn camel:run



Pre requisite: 
—————
For the first time build you will need to internet connectivity so that the maven can download the required jars in the local repo.
